## 如何使用

### 在其他对话中快速调用

```python

# 单次测试
success = run_th2840_test(mode="Cs模式", frequency=100, duration=10)

# 批量测试

```

```python
# 方法2: 直接运行脚本
import subprocess

# 使用默认参数
result = subprocess.run(["python", "th2840_automation.py"], capture_output=True, text=True)

# 指定参数
cmd = [
    "python", "th2840_automation.py",
    "--mode", "Cs模式",
    "--frequency", "1000",
    "--duration", "30",
    "--batch", "4"
]
result = subprocess.run(cmd, capture_output=True, text=True)
```

### 命令行使用

