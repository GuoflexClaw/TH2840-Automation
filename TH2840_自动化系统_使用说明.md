## 快速开始

### 1. 进入文件夹
```bash

```

### 2. 环境检查
```bash

python TH2840_Automation_System/th2840_runner.py --check

# 方法2：进入文件夹后运行
```

### 3. 运行测试
```bash

# 进入文件夹后运行
cd TH2840_Automation_System
python th2840_runner.py --quick
```

## 在其他对话中使用

### 方法1：使用通用运行器（推荐）
```python
import subprocess

# 从任何位置运行
cmd = ["python", "TH2840_Automation_System/th2840_runner.py", "--mode", "Cs模式", "--frequency", "100"]
result = subprocess.run(cmd, capture_output=True, text=True)

if result.returncode == 0:
    print("测试成功")
else:
    print(f"测试失败: {result.stderr}")
```

### 方法2：导入快速调用接口
```python
import sys
import os

# 添加路径并导入
sys.path.append("TH2840_Automation_System")
from th2840_runner import run_th2840_from_anywhere

# 单次测试
success = run_th2840_from_anywhere(mode="Cs模式", frequency=100, duration=10)

# 批量测试
success = run_th2840_from_anywhere(mode="Cp模式", frequency=1000, batch=3)
```

### 方法2：直接运行脚本
```python
import subprocess

# 使用绝对路径
script_path = "TH2840_Automation_System/th2840_automation.py"
cmd = ["python", script_path, "--mode", "Cs模式", "--frequency", "100"]
result = subprocess.run(cmd, capture_output=True, text=True)
```

## 重要提示

1. **首次使用前**：务必运行环境检查 `python th2840_quick_call.py --check`
2. **路径配置**：确保 `th2840_config.json` 中的软件路径正确
3. **屏幕分辨率**：如果屏幕分辨率变化，可能需要重新校准坐标
4. **文件权限**：确保有权限访问保存路径 `E:\A-Openclaw\`
